Here's a code block in Python:

```python
def greet(name):
    print(f"Hello, {name}!")
```

And here's some inline code: `print("Hello, world!")`
You can also have code blocks in other languages, like JavaScript:

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
}
```
